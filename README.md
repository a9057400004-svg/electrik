# Premium electrician site for Vercel

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Upload the project to GitHub.
2. Import the repository into Vercel.
3. In Vercel, open **Settings → Environment Variables**.
4. Add:
   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_CHAT_ID`
5. Deploy.
